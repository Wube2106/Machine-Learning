PROJECT TITLE
Handwritten Digit Recognition using Feedforward Neural Network
DATASET
MNIST dataset (60,000 training + 10,000 testing images)
MODEL ARCHITECTURE
Input Layer: 784 neurons
Hidden Layer: 30 neurons (ReLU)
Output Layer: 10 neurons (Softmax)
HYPERPARAMETERS
Learning rate: 3.0
Epochs: 30
Batch size: 20
Regularization: 5.0
Final Accuracy
94.8% test accuracy
